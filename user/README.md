# FixBuddy User App (Customer Frontend)

The customer-facing application where users can find, book, and track professional services.

## 🚀 Built With
-   **Core**: React.js (Vite)
-   **Styling**: TailwindCSS, Shadcn UI
-   **Navigation**: React Router DOM (v6+)
-   **HTTP Client**: Axios
-   **Real-time Logic**: Socket.io Client
-   **State**: Context API (AuthContext)

## 📦 Project Setup
1.  `npm install`
2.  Create `.env` with `VITE_API_URL`
3.  `npm run dev`

## 🕹 Key Pages
-   **Login & SignUp**: Phone OTP flow.
-   **Dashboard**: Job tracking and quick look.
-   **Find Professionals**: Map and list view of local captains.
-   **Active Job**: Real-time status tracking and chat.
-   **Review**: Feedback system after job completion.
