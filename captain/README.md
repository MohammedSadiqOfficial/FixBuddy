# FixBuddy Captain App (Provider Frontend)

The professional-facing portal for service providers (Captains) to manage jobs, profiles, and earnings.

## 🚀 Built With
-   **Framework**: React.js (Vite)
-   **UI library**: Shadcn UI & TailwindCSS
-   **Navigation**: React Router
-   **Sockets**: Socket.io Client for live job requests and chat
-   **Authentication**: Multi-step OTP flow

## 🛠 Features
-   **Service Management**: Toggle availability and set hourly rates.
-   **Job Feed**: Live job requests based on geolocation.
-   **Active Job Dashboard**: Integrated status updates and customer chat.
-   **Earnings Tracker**: Summary of total and per-job income.
-   **Profile Setup**: Skills, portfolio images, and bio.

## 📦 Setup
1.  `npm install`
2.  Configure `VITE_API_URL` in `.env`
3.  `npm run dev`
