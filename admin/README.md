# FixBuddy Admin Dashboard

The administrative portal for monitoring the entire FixBuddy ecosystem.

## 🚀 Built With
-   **Frontend**: React.js with Vite
-   **Theme Engine**: TailwindCSS v4 with custom HSL palette
-   **Components**: Shadcn/UI (Radix-based)
-   **Architecture**: Tab-based management system

## 🕹 Management Panels
-   **Dashboard Overview**: Vital stats (Active Jobs, Revenue, Top Captains).
-   **User Management**: Monitor customer activity and account status.
-   **Captain Directory**: Verify, approve, and manage service providers.
-   **Request Monitor**: Track all service requests and job lifecycles.

## 📦 Setup
1.  `npm install`
2.  Update `VITE_API_URL` in `.env`
3.  `npm run dev`

---
Admin access requires valid platform credentials.
